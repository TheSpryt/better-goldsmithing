# Better Goldsmithing

A RuneLite plugin for Old School RuneScape.

When smelting gold ore at the Blast Furnace it:

- **Blocks depositing gold ore** onto the conveyor unless your **Gauntlets of goldsmithing** are equipped, so you never lose the bonus XP on a batch.
- **Locks the glove slot** (no equipping or removing gloves) from the moment you deposit until the **Smithing XP for that deposit is credited** — guaranteeing the gauntlets are still on when the bonus lands. After the XP drop you are free to swap to ice gloves.

A safety timer auto-unlocks the slot if no XP arrives, so you can never get stuck.

## Build / run

```
./gradlew run
```
